# Cosmetics Express
RESTful API for Cosmetics Shop 
## Usage
### Install
```
pnpm i
```
### Set environment (.env)
```
PORT=YOUR_PORT
MONGODB_URI=YOUR_MONGODB_URI
DOMAIN=YOUR_DOMAIN_URL
JWT_SECRET=YOUR_JWT_SECRET
```
### Run
```
pnpm run dev
```
